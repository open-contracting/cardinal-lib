Command-line interface
======================

The executable (a.k.a. binary or program) is ``ocdscardinal``.

`ZIP archives of precompiled binaries for OCDS Cardinal are available for Windows, macOS and Linux. <http://github.com/open-contracting/cardinal-rs/releases>`__

If you're a **Rust programmer**, ocdscardinal can be installed with ``cargo``:

.. code:: bash

   cargo install ocdscardinal
